Agregador de noticias sobre GNU/Linux y software libre

Acerca de...

Este sitio quiere retomar el proyecto PlanetaLibre.es creado por el desarrollador y administrador de sistemas Jesús Camacho aka. Zagur. Dicho proyecto quedó discontinuado.

PlanetaLibre es un agregador de noticias de blogs y webs con temáticas relacionadas con el sistema operativo GNU/Linux y/o con software libre. Quiere servir como punto de unión, como ubicación y difusión de dichas webs.

Esta web que lees está creada gracias al código en Python desarrollado por Selairi editor del blog cartaslinux.worpress.coms y disponible GitHub y a partir de ese código crear una página en html5 hospedada utilizando los servicios GitLab Pages.

La recopilación de fuentes y actualización de la página, se realiza todos los días a las 11h y 18h, hora peninsular de España. Se realiza de manera automática mediante una tarea cron habilitada en mi Raspberry Pi, donde se ejecuta el código.

Esta web no tiene analíticas de uso, no guarda logs de visitas ni tiene anuncios. Espero que os guste y que sea un punto de encuentro para las personas que disfruten de GNU/Linux y del software libre o para quienes busquen información sobre dichos temas, ofreciendo enlaces a los blogs y webs que difunden sobre ello.

Si tienes un blog sobre estos temas y quieres que se incluya en la lista de fuentes RSS publicadas manda un correo electrónico.

La imagen de la cabecera está sacada de la web opensource.com/ y tiene licencia CC-by-sa.
